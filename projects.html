ShuffleBite Mobile App and Website

As the team lead and full-stack developer, I oversaw the development of ShuffleBite, an app and website that helps
users find random restaurants. The project was created primarily using React-Native and JavaScript and utilizes the Yelp API
to fetch restaurant information. Bootstrap was used for the front-end and node.js was utilized in order to implement a proxy
server used by the website for API calls.

Line Following and Object Detecting Robot

Using a Raspberry Pi Pico, I developed a line following robot that also used object detection in order to avoid
collisions utilizing a variety of sensors. The entire program was made using Python and external libraries that allowed
me to utilize the hardware.

Word Guess Game

This command line game was made using Python, a random word API, and a dictionary API. It improves upon previously
created word guessing games by introducing a wider variety of word choices with random lengths, which are all collected
from the random word API. The dictionary API is then used to verify inputs, bringing everything together to make a fun,
yet challenging, game that elevates the user experience.
